# se5techcalc
This is a simple calculator for determining how many percentage points to allocate when researching multiple tech levels in Space Empires 5.

![example](/example.png)

Values from the ingame research screen will need to be manually copied into the calculator to get a result.
To help with this, there is a function to import the game's TechAreas.txt file, which will autofill the cost per level and allow tech areas to be selected by typing or clicking the dropdown menu.
This should work with mods as well, as long as the correct definition file is imported.

There is a HELP button with instructions for using the calculator for any new players of SE5.

Python must be installed on your machine to run this.
